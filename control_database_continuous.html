control_database_continuous module¶

This program takes the FFHQ feature metadata, extracts the pose of the face and looks for a threshold to predict if an image will be well detected based on the pose

control_database_continuous.main()[source]¶

Run program:

Stars by loading feature FFHQ dataset

Then calculates how many images were correctly detected

Then extracts the headpose metadata

Finally plots the headpose angle discriminating which were detected or not
